## 修正手法

### ファイル編集の戦略
1. 最初は Edit で小さい修正（Header, Stage 1 ゲート等）を試行
2. 韓国語混在が大量に判明 → 全文 Write に切り替え（440+ 行のため）
3. Read では正確な行番号が必要なため、まず全体 Read → エスケープ問題の確認

### 韓国語エスケープの問題
- 韓国語テキスト（예, 각, 담당, 상태, 속성 等）はシステム側が Unicode として処理
- Edit の old_string に含めようとすると、エスケープ表現と実テキストの不一致が発生
- 対策: 全文 Write で一度にリプレイス
